# Poddavki (Giveaway Checkers)

Classic Poddavki (Giveaway Checkers) game implementation in Python.

## Rules
- Player with NO valid moves WINS
- Mandatory captures: if capture available, must capture
- Regular pieces move forward only
- Kings move forward/backward
- Goal: Lose all pieces or get blocked

## Requirements
- Python 3.x
- Pygame

## Installation
pip install pygame

## Run
python app.py

## Controls
- Mouse: Select pieces and make moves
